# Staff_Management_System_SimpleProject
A simple Staff Management System created using Spring Boot and RESTful API. This application has CRUD operations that help in efficient management of staff details. This project is an example of backend development using Spring Boot. 🚀

**Description**
This is a simple Staff Management System developed using Spring Boot and RESTful APIs.
The project allows users to manage staff information by performing basic CRUD operations such as adding, updating, deleting, and viewing staff details.
It is designed as a beginner-friendly backend project to demonstrate how REST APIs work in Spring Boot.
The system follows a clean architecture and helps in understanding API development and database integration.

**Features**
Add new staff details
View all staff records
Update existing staff information
Delete staff records
RESTful API based backend

**Technologies Used**
Java
Spring Boot
RESTful APIs
Maven
MySQL / H2 Database

**API Endpoints**
GET /staff – Get all staff details
GET /staff/{id} – Get staff by ID
POST /staff – Add new staff
PUT /staff/{id} – Update staff details
DELETE /staff/{id} – Delete staff

**Project Purpose**
This project was created for learning Spring Boot and REST API development and to understand backend application structure.
